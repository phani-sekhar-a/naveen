MediInternia is a PHP-based internship management website deployed online.
The system operates with a remote MySQL database and runs automatically once hosted on a PHP-supported server.


 Website Deployment (Developer Setup)

The following steps are used to deploy MediInternia on any PHP hosting platform:

1️ Upload all project files to the hosting server
(inside the web root such as public_html/)

2️ Directly access the index page for the preview

3 no database setup required as it is remote
